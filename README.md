# app-stocks-portfolio
App developed in Python for users' stocks portfolio.
This app is a CRUD that manages data on MongoDB cloud database at Atlas.com.

Technologies:
  - Python, Flask
  - MongoDB Cloud
  - Docker
  - Linux
